{# Icon set — Lucide icons vendored as Jinja macros. Each icon is a small parameterized macro. The caller passes a Tailwind `class` string for size + color (using `currentColor`), and may override `stroke_width` if the design calls for a heavier or lighter stroke than the Lucide default (2). SVG markup is build-time inlined. Usage ----- {% import 'macros/icons.html' as icons %} {{ icons.check(class='size-3 text-shovels-primary', stroke_width=3) }} {{ icons.chevron_down(class='size-5 text-gray-400') }} Adding a new icon ----------------- 1. Go to https://lucide.dev/icons/ 2. Find the icon, click "Copy SVG" 3. Paste the inner SVG content into a new macro below 4. Replace any hard-coded class with `class="{{ class }}"` 5. Replace any hard-coded stroke-width with `stroke-width="{{ stroke_width }}"` 6. Keep `aria-hidden="true"` on purely decorative icons; remove it and add a `` element if the icon conveys meaning independent of nearby text.
