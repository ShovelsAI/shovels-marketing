Each renders with a green checkmark. image_src : Path to the screenshot. image_alt : Alt text for the screenshot. caption : Optional small italic caption rendered below the screenshot (e.g., "names, and dates"). The first case displays image-right; the second image-left; alternating thereafter. Alternation is handled internally via loop.index.
